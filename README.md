## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/thayoob/Supported-Note.git
   cd Supported-Note
   ```

2. Install dependencies:

   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```
   _(Use `npm start` if using Create React App)_
